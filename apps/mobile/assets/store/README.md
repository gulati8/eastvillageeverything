## Apple App Store Required Assets

| Asset | Dimensions | Format | Notes |
|---|---|---|---|
| App icon | 1024 x 1024 px | PNG, no alpha | Required; Xcode generates smaller sizes |
| iPhone 6.5" screenshot | 1242 x 2688 px (or 1284 x 2778) | PNG or JPEG | Minimum 3, max 10 |
| iPhone 5.5" screenshot | 1242 x 2208 px | PNG or JPEG | Required for older device support |
| iPad 12.9" screenshot | 2048 x 2732 px | PNG or JPEG | Required if iPad supported |
| Preview video (optional) | Up to 30 s, 15–500 MB | .mov / .mp4 | 1920 x 1080 or portrait equivalent |

Reference: https://developer.apple.com/help/app-store-connect/reference/screenshot-specifications

---

## Google Play Store Required Assets

| Asset | Dimensions | Format | Notes |
|---|---|---|---|
| High-res icon | 512 x 512 px | 32-bit PNG | Displayed on Play Store listings |
| Feature graphic | 1024 x 500 px | PNG or JPEG | Banner shown at top of store listing |
| Phone screenshot | 320–3840 px on any side, 2:1 max ratio | PNG or JPEG | Minimum 2, max 8 |
| 7-inch tablet screenshot | 320–3840 px | PNG or JPEG | Optional but recommended |
| 10-inch tablet screenshot | 320–3840 px | PNG or JPEG | Optional but recommended |
| Promo video (optional) | YouTube URL | n/a | Optional feature graphic overlay |

Reference: https://support.google.com/googleplay/android-developer/answer/9866151
