Admin enriched content mockups

Three proposals for the place editor Nicholas will actually use.

These mockups keep the old web directory safe while preparing content for the richer mobile app. Images are upload/camera/library only, resized and cropped by the system. AI text is generated into editable fields, never auto-published.

Image input Required

Take photo, choose from library, or upload file. No URL field in the editor.

Image processing Phase 2

Crop presets for list thumbnail and detail hero. Save processed variants.

AI text Phase 2

Generate pitch, perfect-for, insider tip, vibe, and crowd. Nicholas edits before save.

Compatibility Always

New fields stay optional. Existing web and mobile must fall back gracefully.
